DNA to protein converter | Momchi's Genetics Tools

Convert DNA sequences into protein chains.

Output Protein Sequence:

This tool translates DNA sequences into corresponding polypeptide chains. Each amino acid is represented using the standard one-letter code. Translation begins at the first nucleotide of the input sequence and continues through the entire DNA strand. When a stop codon (TAA, TAG, or TGA) is encountered, it is represented in the output as an underscore (_). The tool supports DNA sequences written in either uppercase (A, T, G, C) or lowercase (a, t, g, c), and can also process sequences in FASTA format.
